Kebijakan Privasi — Umandana Translator

Terakhir diperbarui: 14 November 2025

Terima kasih telah menggunakan Umandana Translator. Privasi dan keamanan data Anda penting bagi kami. Dokumen ini menjelaskan jenis data yang dikumpulkan oleh aplikasi ini, bagaimana data diproses, serta pilihan dan hak Anda.

1. Ringkasan Singkat

Umandana Translator dirancang untuk berfungsi sepenuhnya di perangkat Anda. Secara default aplikasi tidak mengunggah atau menyimpan foto, teks, atau file Anda ke server pihak ketiga. Semua proses konversi/terjemahan dilakukan secara lokal.

2. Data yang Dikumpulkan

Tidak dikumpulkan: Konten teks yang Anda masukkan, foto, atau file lain.

Data opsional: Jika Anda mengaktifkan fitur analytics, aplikasi dapat mengumpulkan data anonim seperti penggunaan fitur dan error teknis untuk meningkatkan kualitas aplikasi. Data ini tidak berisi informasi pribadi.

3. Penggunaan Data

Jika analytics diaktifkan, data anonim digunakan untuk:

  • Meningkatkan performa dan stabilitas aplikasi.
  • Mendapatkan statistik umum penggunaan fitur.

4. Penyimpanan Lokal

Aplikasi dapat menyimpan preferensi seperti tema dan pengaturan pada perangkat Anda melalui localStorage. Data ini tidak dikirim ke server.

5. Layanan Pihak Ketiga

Jika Anda mengaktifkan analytics, data anonim dapat diproses oleh penyedia layanan pihak ketiga seperti Firebase Analytics. Penggunaan layanan tersebut tunduk pada kebijakan privasi penyedia.

6. Keamanan

Kami menerapkan langkah standar untuk menjaga keamanan data lokal Anda. Anda bertanggung jawab menjaga keamanan perangkat Anda.

7. Hak dan Pilihan Anda

Anda dapat:

  • Menonaktifkan analytics kapan saja.
  • Menghapus data lokal dengan menghapus data aplikasi atau melalui pengaturan browser.

Lihat Syarat & Ketentuan untuk aturan penggunaan aplikasi.
